# DDRec

PyTorch implementation for IEEE Transactions on Computational Social Systems: [ [DDRec: Dual Denoising Multimodal Graph Recommendation](https://doi.org/10.1109/TCSS.2024.3490801)]

- Python 3.8
- torch==1.11.0
